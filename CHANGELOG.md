# 更新日志

## v1.1.0

重构 Afdian 插件以使用结构化配置系统和共享数据库实例，并添加用于管理通知会话的管理员命令。

新功能：

- 增加强类型的插件配置层，用于暴露 webhook、API、支付以及通知会话相关的配置项。
- 新增管理员命令，用于将当前会话注册为 Afdian 赞助通知的接收方。

增强改进：

- 重构 Afdian 插件、webhook 服务器和 API 客户端，使其从新的 `PluginConfig` 中加载所有设置，而不是使用临时的字典配置。
- 通过可复用的 `OrderDB` 实例在插件与 webhook 服务器之间共享订单持久化逻辑，实现集中化管理。
- 改进生命周期管理，在插件的 initialize/terminate 钩子中初始化并关闭 webhook 服务器和 API 客户端。
- 清理数据库和 webhook 处理代码中的类型标注与格式，以提升可读性和类型检查器的兼容性。
